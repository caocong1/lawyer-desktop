服务于律师真实工作流

墨律不把文书生成当作孤立按钮，而是把材料、检索、引用、风险提示和反馈运营串成可追踪的闭环。

本地优先

会话、材料、API Key 和反馈状态保存在本地数据库，文件访问走 Tauri 沙箱命令。

法规/案例检索

接入本地法库和 MCP 检索源，输出中保留待核验状态和来源线索。

文书起草

支持合同、诉状、法律意见书、尽调报告、律师函等常用中文法律文书。

引用核验

把法规、案例和风险条款拆成可检查的引用面板，便于律师复核。

DOCX 导出

结构化文书可进入预览区并导出为可继续编辑的办公文档。

反馈闭环

点赞、点踩、维度标签和说明进入 outbox，供团队运营和 skill 改进使用。

Skill 更新

同步服务分发 stable/beta skill 包，让法律能力可以独立于客户端迭代。
